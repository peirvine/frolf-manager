## Installation
 
Install Firebase with npm

```bash
  npm install firebase
```
    
## Run Locally

Clone the project

```bash
  // DO NOT FORK
  git clone git@github.com:peirvine/frolf-manager.git
```

Go to the project directory

```bash
  cd frolf-manager
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run start
```
*Be careful, currently local hits prod data*
